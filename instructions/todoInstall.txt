Logiciels a installer:
- Nodejs
- MongoDB
- Git (Windows: Add git to path)

- SublimeText
- Smartgit
- Robomongo

Initialisation de l'environnement:
- npm install -g grunt
- npm install -g bower
- npm install 
- Bower install

- install de imageMagick (Windows: Add imagemagick to path)
- install de quickthumb

Créer le répertoire img/users dans server/public/ pour l'upload des fichiers.
Créer les répertoire .cache/crop/100x100/img/users et .cache/crop/300x300/img/users dans server/public

Enjoy it!
